Condition :
Write a program that outputs a message based on the type of weather entered:

If sunny, output: Wear sunglasses.
If rainy, output: Take an umbrella.
If snowy, output: Wear warm clothes.
Otherwise, output: Unknown weather type.

use if / else if / else Conditions


### Examples of Input and Output for the Weather Task

#### Example 1:
Input:  
let weather = "sunny";

Output:  
Wear sunglasses.


#### Example 2:
Input:  
let weather = "foggy";

Output:  
Unknown weather type.
